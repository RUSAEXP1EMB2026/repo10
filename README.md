# Fourune Teller (占いシステム)

![Fourune Teller](https://imgur.com/a/iyB7IN1)

このプロジェクトは、Nature Remo 3 のセンサデータと天気予報 API を連携し、ユーザに当日の占い結果を生成して LINE で通知するシステムです。

## 概要

- ユーザは生年月日と占いを受け取りたい時刻を登録する。
- システムは Remo 3 から室内の温度・湿度・明るさ・人感センサデータを取得する。
- 天気予報 API から当日の降水確率を取得する。
- 取得データを Google スプレッドシートに 30 分ごとに記録する。
- その日の占い結果として、次を生成する。
  - 順位（星座×干支で 1〜144 位）
  - 幸福度（0〜100 の百分率）
  - ラッキーアイテム
  - アドバイス

## 主要機能

- 生年月日から星座と干支を自動判定
- Nature Remo 3 のセンサデータ取得
- 天気予報 API から降水確率取得
- 幸福度計算
- ラッキーアイテムとアドバイスの生成
- LINE 連携による通知
- Google スプレッドシートへのデータ記録

## 幸福度計算の概要

幸福度 `H` は次の要素を組み合わせて計算される設計です。

- 星座順位+干支順位 `A`
- 室内環境評価 `B`
- 睡眠時間評価 `C`
- 降水確率評価 `D`

例: `H = 100 × (0.55A + 0.25B + 0.20C + 0.15D + 0.10E)`

## 想定ユーザ

- 占いを楽しみたい人
- その日の気分や行動の目安を簡単に知りたい人
- 室内環境や天候に左右される体調・気分を把握したい人
